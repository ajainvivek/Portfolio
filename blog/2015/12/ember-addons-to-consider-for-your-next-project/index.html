Ember Addons To Consider For Your Project

What are ember addons?

An easy way to package and share common code within the ember ecosystem.

Essential addons for your project

  • ember-moment – score 10

    • Moment.js template helper for ember to Parse, validate, manipulate, and display dates in JavaScript.
      ember install ember-moment

  • ember-async-button – score 9

    • When running async actions ensuring disabling of the button, re-enabling, and handling promise rejections is pretty boilerplate. This component packages up that behavior.
      ember install ember-async-button

  • ember-cli-notifications – score 6

    • Build process hook for notifying messages for ember-cli.
      ember install ember-cli-notifications

  • ember-undo-stack – score 8

    • A simple undo/redo stack for Ember.js objects.
      ember install ember-undo-stack

  • ember-cli-accounting – score 8

    • Accounting.js template helper for ember, which provides simple and advanced number, money and currency formatting.
      ember install ember-cli-accounting

  • ember-cli-mirage – score 10

    • A client-side mock HTTP server to develop, test and demo your Ember app.
      ember install ember-cli-mirage

  • ember-cli-babel – score 10

    • This Ember-CLI plugin uses Babel to allow you to use ES6 syntax with your Ember-CLI project.
      ember install ember-cli-babel

  • ember-watson – score 10

    • A young Ember Doctor to help you fix your code or cleans your code.
      ember install ember-watson

  • ember-cli-htmlbars – score 9

    • A library for adding htmlbars to ember CLI. htmlbars is a variant of Handlebars that emits DOM and allows you to write helpers that manipulate live DOM nodes.
      ember install ember-cli-htmlbars

  • ember-cli-sass – score 8

    • Use node-sass to preprocess your ember-cli app’s files, with support for sourceMaps and include paths.
      ember install ember-cli-sass

  • ember-lodash – score 9

    • Lodash in ES6 for Ember.js apps. Lodash provides a whole mess of useful functional programming helpers without extending any built-in objects.
      ember install ember-lodash

Discovering Ember Addons

  • EmberAddons.com
  • EmberObserver.com

EmberObserver is a better way to discover and compare ember-cli addons. Before installing any addon go ahead and see the EmberObserver score.
